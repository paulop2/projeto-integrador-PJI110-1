**Common failure patterns and fixes:**

1. `ImportError: cannot import name X from src.admin` — admin module file missing or has syntax error. Read the error, fix the import.

2. `AssertionError: 404 != 201` on POST /admin/alunos — router not registered in main.py. Check that `app.include_router(admin_router)` is in `backend/src/main.py`.

3. `AssertionError: 422 != 201` — request body schema mismatch. Check that the test JSON matches the Pydantic schema in admin/schemas.py.

4. `IntegrityError: UNIQUE constraint failed: professor_turma` — sync_professor_turma delete not running before insert. Check `_sync_professor_turma` in service.py uses `synchronize_session=False`.

5. `test_admin_self_deactivation_blocked FAILED: assert 200 in (400, 403)` — deactivate_usuario endpoint not checking caller_id. Check `service.deactivate_usuario` has the `caller_id == target_id` guard.

6. `AssertionError: 'matricula' not in response.json()` — Aluno model missing matricula, or migration 0003 not applied. Run `alembic current` to verify 0003 is head.

After all fixes, re-run until exit code is 0 with all 11 tests passing.
