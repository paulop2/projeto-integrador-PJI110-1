3. Create `backend/src/password_reset/service.py` with these functions:

`generate_reset_token(db: Session, email: str) -> str | None`:
- Query Usuario by email (case-insensitive); if not found return None (don't reveal)
- If not user.ativo: return None (treat as not found)
- DELETE existing unused tokens for this user: `db.query(ResetToken).filter_by(usuario_id=user.id, usado=False).delete()`
- Generate token: `secrets.token_urlsafe(32)` — 43 chars URL-safe
- expira_em = datetime.now(timezone.utc) + timedelta(hours=24)
- Insert new ResetToken(usuario_id=user.id, token=token, expira_em=expira_em, usado=False)
- db.commit()
- Return token string

`send_reset_email(to_email: str, token: str) -> None`:
- Build reset URL: `f"{settings.FRONTEND_URL}/redefinir-senha?token={token}"`
- Build email using email.mime.multipart.MIMEMultipart + email.mime.text.MIMEText:
  - Subject: "Redefinição de senha — Sistema Escolar"
  - Body (plain text): Portuguese message explaining the link expires in 24 hours
  - Include the full reset URL in the body
- Send via smtplib.SMTP(settings.SMTP_HOST, settings.SMTP_PORT):
  - server.starttls()
  - server.login(settings.SMTP_USER, settings.SMTP_PASS)
  - server.sendmail(settings.SMTP_SENDER, to_email, msg.as_string())
- If SMTP_USER is empty string (not configured), skip sending and log to stdout: `print(f"[DEV] Reset link: {reset_url}")` — this allows testing without Mailtrap config

`validate_and_consume_token(db: Session, token: str) -> Usuario | None`:
- Query ResetToken by token string
- If not found: return None
- If token.usado: return None
- If token.expira_em < datetime.now(timezone.utc): return None
- Mark token.usado = True
- db.commit()
- Return db.get(Usuario, reset_token.usuario_id)

`update_password(db: Session, user: Usuario, nova_senha: str) -> None`:
- Import hash_password from src.auth.service
- user.senha_hash = hash_password(nova_senha)
- db.commit()

1. Create `backend/src/password_reset/router.py`:
```python
from fastapi import APIRouter, Depends, HTTPException, status
from sqlalchemy.orm import Session
from src.database import get_db
from src.password_reset.schemas import ForgotPasswordRequest, ForgotPasswordResponse, ResetPasswordRequest, ResetPasswordResponse
from src.password_reset.service import generate_reset_token, send_reset_email, validate_and_consume_token, update_password
from src.auth.service import create_access_token, get_display_name

router = APIRouter(tags=["password-reset"])

@router.post("/forgot-password", response_model=ForgotPasswordResponse)
def forgot_password(body: ForgotPasswordRequest, db: Session = Depends(get_db)):
    token = generate_reset_token(db, body.email)
    if token:
        send_reset_email(body.email, token)
    # Always return the same message — do NOT reveal if email exists
    return ForgotPasswordResponse(
        message=f"Se este email estiver cadastrado, você receberá um link de redefinição em breve."
    )

@router.post("/reset-password", response_model=ResetPasswordResponse)
def reset_password(body: ResetPasswordRequest, db: Session = Depends(get_db)):
    if len(body.nova_senha) < 8:
        raise HTTPException(status_code=400, detail="Senha deve ter pelo menos 8 caracteres")
    user = validate_and_consume_token(db, body.token)
    if not user:
        raise HTTPException(status_code=400, detail="Token inválido ou expirado")
    update_password(db, user, body.nova_senha)
    # Auto-login: issue new JWT
    access_token = create_access_token({"sub": str(user.id), "tipo": user.tipo.value})
    nome = get_display_name(db, user)
    return ResetPasswordResponse(
        access_token=access_token,
        token_type="bearer",
        user={"id": user.id, "email": user.email, "tipo": user.tipo.value, "nome": nome},
    )
```
