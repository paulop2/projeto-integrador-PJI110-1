  <action>
Start both servers (in background or separate terminals):

```bash
# Terminal 1 — backend
cd backend && uvicorn src.main:app --reload

# Terminal 2 — frontend
cd frontend && npm run dev
```

Run automated API checks:

```bash
# 1. Verify backend health
curl -s http://localhost:8000/health | python -m json.tool

# 2. Login with admin credentials
curl -s -X POST http://localhost:8000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@escola.dev","senha":"Admin@123"}' | python -m json.tool

# 3. Invalid credentials → expect 401
curl -s -o /dev/null -w "Status: %{http_code}\n" \
  -X POST http://localhost:8000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@escola.dev","senha":"wrongpassword"}'

# 4. Protected endpoint without token → expect 401
curl -s -o /dev/null -w "Status: %{http_code}\n" \
  http://localhost:8000/auth/me

# 5. Forgot password endpoint
curl -s -X POST http://localhost:8000/auth/forgot-password \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@escola.dev"}' | python -m json.tool

# 6. Forgot password with unknown email — same 200 response (no info leak)
curl -s -X POST http://localhost:8000/auth/forgot-password \
  -H "Content-Type: application/json" \
  -d '{"email":"naocadastrado@test.com"}' | python -m json.tool
```

Report any failures. If all pass, continue to checkpoint.
