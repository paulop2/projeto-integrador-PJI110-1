2. Create `backend/src/auth/schemas.py`:
```python
from pydantic import BaseModel, EmailStr

class LoginRequest(BaseModel):
    email: EmailStr
    senha: str

class UserInfo(BaseModel):
    id: int
    email: str
    tipo: str
    nome: str

class LoginResponse(BaseModel):
    access_token: str
    token_type: str = "bearer"
    user: UserInfo
```

3. Create `backend/src/auth/service.py`:
- Import: jwt, datetime/timedelta/timezone, CryptContext from passlib.context, settings from src.config, Session from sqlalchemy.orm, Usuario/ResetToken/TipoUsuario from src.models.usuario
- `pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")`
- `verify_password(plain: str, hashed: str) -> bool` — uses pwd_context.verify
- `hash_password(plain: str) -> str` — uses pwd_context.hash
- `create_access_token(data: dict) -> str`:
  - expire = datetime.now(timezone.utc) + timedelta(days=settings.ACCESS_TOKEN_EXPIRE_DAYS)
  - to_encode = {**data, "exp": expire}
  - return jwt.encode(to_encode, settings.SECRET_KEY, algorithm="HS256")
- `authenticate_user(db: Session, email: str, senha: str) -> Usuario | None`:
  - query Usuario by email (case-insensitive: filter by email.lower())
  - if not found: return None
  - if not user.ativo: return None
  - if not verify_password(senha, user.senha_hash): return None
  - return user
- `get_display_name(db: Session, user: Usuario) -> str`:
  - import Professor, Responsavel inline (to avoid circular imports) from src.models if those models exist — but Phase 1 may not have SQLAlchemy models for all tables. Use raw SQL instead:
    - if user.tipo.value == "professor": query `SELECT nome FROM professores WHERE usuario_id = :uid`
    - elif user.tipo.value == "responsavel": query `SELECT nome FROM responsaveis WHERE usuario_id = :uid`
    - else (admin): return "Administrador"
  - Return profile nome if found, else user.email as fallback
- `maybe_renew_token(user: Usuario) -> str | None`:
  - Decode current token to get exp; if exp - now < 86400s, return new token
  - Actually: this function accepts the raw token string, decodes it, checks expiry, returns new token string if needed
  - Signature: `maybe_renew_token(token: str) -> str | None`
  - Return None if not near expiry, new token string if exp within 24h

4. Create `backend/src/auth/dependencies.py`:
- `oauth2_scheme = OAuth2PasswordBearer(tokenUrl="/auth/login")`
- `get_current_user(token: Annotated[str, Depends(oauth2_scheme)], db: Session = Depends(get_db)) -> Usuario`:
  - try: payload = jwt.decode(token, settings.SECRET_KEY, algorithms=["HS256"])
  - except InvalidTokenError: raise HTTPException(401, "Token inválido", {"WWW-Authenticate": "Bearer"})
  - user_id = payload.get("sub"); if None: raise 401
  - user = db.get(Usuario, int(user_id)); if None or not user.ativo: raise 401
  - return user
- `require_role(role: str)` factory function:
  - returns checker function: `def checker(user: Usuario = Depends(get_current_user))` that raises 403 if user.tipo.value != role

5. Create `backend/src/auth/router.py`:
- `router = APIRouter(prefix="/auth", tags=["auth"])`
- `POST /login` endpoint:
  - accepts LoginRequest body (JSON — NOT OAuth2PasswordRequestForm)
  - calls authenticate_user; if None raises HTTPException(401, "Email ou senha incorretos")
  - calls create_access_token({"sub": str(user.id), "tipo": user.tipo.value})
  - calls get_display_name to resolve nome
  - returns LoginResponse(access_token=token, token_type="bearer", user=UserInfo(id, email, tipo, nome))
- `GET /me` endpoint (protected, for testing):
  - Depends(get_current_user), returns basic user info

6. Update `backend/src/main.py`:

- Add: `app.include_router(auth_router)` AFTER the CORS middleware setup
