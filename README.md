# Sistema de Registro de Presenca e Notas

Sistema web para registro de presenca e notas de alunos, desenvolvido como Projeto Integrador (PJI110) - Turma 19.

## Descricao

Sistema com separacao frontend/backend contendo tres tipos de acesso:
- **Admin**: Gerenciamento completo do sistema
- **Professor**: Registro de chamadas e lancamento de notas
- **Responsavel**: Consulta de boletim e frequencia dos alunos vinculados

## Stack Planejada

- **Backend**: Python (FastAPI)
- **Frontend**: React
- **Banco de Dados**: SQLite
